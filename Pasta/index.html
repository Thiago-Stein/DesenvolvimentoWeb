Como acessar os livros:

Para acessar os livros de forma gratuita eu estarei disponibilizando links para a Biblioteca Elfica!

Porém, para acessar a biblioteca élfica da "Melhor forma possível" siga esses passos:

  1. Copie esse link https://bibliotecaelfica.org/
  2. Acesse o site https://web.archive.org/
  3. Cole o link da biblioteca élfica no campo de busca do site
  4. Escolha uma data para acessar o site, de preferência sendo do começo de 2024 que eu garanto que funciona
  5. Pronto, agora é só acessar os livros de graça por lá!

Fazer isso garante que o download dos livros será possível, porém é para os links disponibilizados na página respectiva ao sistema funcionarem!
